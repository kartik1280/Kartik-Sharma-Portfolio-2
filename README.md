### Flexbox Usage
- Header navigation (logo + links)
- Hero section (text + image side-by-side)
- About section layout

### CSS Grid Layouts
- Skills: 6 cards in responsive grid
- Projects: 4 project cards with hover effects

### Responsive Breakpoints Tested
- Desktop: >768px (Full layout)
- Tablet: 768px (Stacked hero, adjusted spacing)
- Mobile: <480px (Single column grids, stacked nav)


### Animations & Effects
- Hero text: Color-shifting "Full Stack Developer" (@keyframes colorShift)
- Fade-in animations on scroll (@keyframes fadeInUp)
- Hover transforms on cards/buttons (translateY, scale)
- Smooth transitions (0.3s ease)

### Relative Units Used
- rem for spacing/fonts
- % for widths
- vw/vh for hero sizing
- clamp() for responsive typography
- grid minmax() for cards

## Tech Stack
HTML5 | CSS3 | Flexbox | CSS Grid | Media Queries | Keyframe Animations
Google Fonts (Poppins) | Mobile-First Design


## How to Run
1. Clone repo
2. Open `index.html` in browser
3. Resize window to test responsive behavior
